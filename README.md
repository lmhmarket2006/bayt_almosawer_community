\# Bayt Almosawer Community



منصة مجتمع خاصة بخريجي ومتدربي أكاديمية بيت المصور.



\## الفكرة



مجتمع اجتماعي مهني للمصورين يشبه خليطًا بين Facebook وInstagram وX، لكنه مخصص لخريجي ومتدربي الأكاديمية لعرض الأعمال، التعليقات، الإعجابات، التقييمات، وبناء بروفايل مهني لكل مصور.



\## المميزات الحالية



\- تسجيل حساب جديد

\- تسجيل دخول وخروج

\- موافقة الإدارة على الأعضاء

\- أدوار المستخدمين: Admin / Trainer / Member

\- لوحة إدارة المستخدمين

\- إنشاء منشورات

\- رفع صور للمنشورات

\- لايكات

\- تعليقات

\- حذف التعليقات حسب الصلاحية

\- بروفايل لكل مصور

\- تعديل الملف الشخصي

\- رفع صورة شخصية وصورة غلاف

\- تصميم عربي RTL

\- Mobile First UI



\## التقنيات



\- Flask

\- SQLAlchemy

\- Flask-Login

\- SQLite محليًا

\- PostgreSQL عند النشر

\- Bootstrap RTL

\- Custom CSS

\- Railway-ready
